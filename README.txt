===========================================
chapter-1到chapter-16目錄分別包含了本書第1章到第16章所用到的原始程式碼，以及部分插圖。
原始程式碼在src子目錄中，按照Android原始程式碼專案目錄結構來組織，包含了Android系統的原始程式碼以及各範例APP的原始程式碼。
在第7章、第8章、第9章、第10章、第14章和第16章中，有部分過程圖（序列圖）由於過大，被分割成了若干個子過程圖來描述，為了方便讀者理解這些過程圖，chapter-7、chapter-8、chapter-9、chapter-10、chapter-14和chapter-16目錄將這些被分割了的子過程圖合成了一個完整的過程圖。
